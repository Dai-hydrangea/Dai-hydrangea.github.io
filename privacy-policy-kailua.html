Kailua プライバシーポリシー

最終更新日: 2026-05-08

はじめに

Kailua (以下、 「本アプリ」) は、 ユーザーの自己分析・自己観察を支援する iOS アプリケーションです。 本アプリの開発者 (以下、 「開発者」) は、 ユーザーのプライバシーを尊重し、 収集・送信される情報の取り扱いについて、 以下の方針に従って適切に管理します。

1. 基本機能で収集する情報

本アプリの無料機能 (シナリオ選択・自己観察結果・8 軸分析・MBTI 推定・本棚提案など) は、 すべて端末内で完結します。 これらの機能では、 以下の情報を 一切外部に送信しません:

  • 氏名・メールアドレス・電話番号などの個人情報
  • 位置情報
  • 連絡先・写真・カレンダーなどの端末内データ
  • 利用ログ・解析データ・広告 ID

2. 端末内に保存される情報

本アプリは、 ユーザーの選択履歴 (どの選択肢を選んだか、 何秒考えたか、 等) を端末内のみに保存します。 これらのデータは、 ユーザー自身の自己観察のために利用され、 外部サーバーに送信されることはありません。 アプリを削除すると、 これらのデータも削除されます。

3. Pro 版 AI 深掘り対話機能 (有料機能)

本アプリの Pro / Super Pro プラン (有料サブスクリプション) をご利用の場合、 「AI と深掘りする」 機能を使うと、 入力したテキストおよび自己観察の文脈情報が外部の AI サービスに送信されます。 本機能は 有料機能であり、 ユーザーが明示的に利用を選択した場合のみ動作します。 無料プランで本機能を利用することはできません。

3-1. 送信される情報

ユーザーが AI 深掘り対話機能を利用した際、 以下の情報が外部に送信されます:

  • ユーザーが対話画面に入力したテキスト
  • 本アプリ内で実施した自己観察の結果 (MBTI 推定タイプ、 8 軸スコア、 シナリオごとの選択履歴 + 選択所要時間)
  • 端末識別子 (iOS の identifierForVendor、 月次の利用回数管理目的のみで利用)

氏名・メールアドレス・電話番号・位置情報・連絡先などの個人情報は 送信されません。 送信される自己観察情報は本アプリ内で生成されたものに限られます。

3-2. 送信経路 (中継サーバー)

ユーザーの入力は、 開発者が運用する Cloudflare Workers 上の中継サーバー を経由して、 Anthropic 社の Claude API に転送されます。 本中継サーバーは API キーを安全に保持し、 端末からのリクエストを認証する目的のみに使用されます。

  • 送信内容は中継サーバーに永続的に保存されません
  • 月次利用回数管理のため、 端末識別子と月の利用回数のみ Cloudflare Workers KV に最大 32 日間保存され、 自動削除されます
  • 中継サーバーは Cloudflare の Edge ネットワーク上で動作します (運用者: 開発者)

3-3. Anthropic 社による処理

中継サーバーから転送された情報は、 Anthropic, PBC が提供する Claude API (claude-opus-4-7 等) によって処理され、 対話の応答が生成されます。 Anthropic 社のデータ取り扱いについては、 同社のプライバシーポリシーをご参照ください:
https://www.anthropic.com/privacy

本アプリは、 Anthropic API への送信時に Prompt Caching 機能を利用しています。 Anthropic 社は API リクエストデータをモデルの学習に利用しないことを明示しています (上記プライバシーポリシー参照)。

3-4. ユーザーの選択肢

  • 本機能を 利用しない選択 はいつでも可能です (無料プランのまま自己観察結果のみご利用ください)
  • サブスクリプションは App Store の登録ページからいつでも解約できます
  • 対話履歴の端末内保存については、 アプリを削除すると共に削除されます

4-1. Apple App Store

本アプリは Apple App Store を通じて配信されています。 アプリのダウンロード・サブスクリプション購入・利用統計などは、 Apple のプライバシーポリシーに従って Apple 社で処理されます。

4-2. Amazon アソシエイト・プログラム

本アプリには、 ユーザーへの書籍推薦機能があります。 推薦された書籍のリンクをタップすると、 Amazon の商品ページが開きます。 これらのリンクは、 Amazon アソシエイト・プログラム (アソシエイト ID: daihydrangea-22) によるアフィリエイトリンクです。 ユーザーがリンク経由で商品を購入した場合、 開発者は Amazon から紹介料を受け取ることがあります。 リンクをタップすると、 Amazon のプライバシーポリシーに従って Amazon 側で情報が処理されます。

Amazon のプライバシーポリシー: https://www.amazon.co.jp/gp/help/customer/display.html?nodeId=GX7NJQ4ZB8MHFRNJ

5. 性格タイプ分類について

本アプリは、 16 タイプの性格分類 (内向/外向、 直観/感覚、 思考/感情、 判断/知覚) に基づく、 一般的な自己観察ツールです。 公式の心理テストではありません。 本アプリの結果は、 専門的な診断や医療目的での使用を意図したものではありません。

MBTI® は The Myers-Briggs Company の登録商標です。 本アプリは MBTI 公式テストとは一切関係ありません。

6. お子様の利用について

本アプリは 4 歳以上の利用を想定しています。 お子様が利用される場合は、 保護者の方の同意のもとでご利用ください。 無料機能では個人情報の収集は一切行いません。 Pro 版の AI 深掘り対話機能は、 ユーザー自身が入力したテキストを外部に送信するため、 お子様が利用する場合は保護者の判断のもとでご利用ください。

7. プライバシーポリシーの変更

本ポリシーは、 必要に応じて更新されることがあります。 重要な変更がある場合は、 本ページにて告知します。

8. お問い合わせ
