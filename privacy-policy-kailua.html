はじめに

Kailua (以下、 「本アプリ」) は、 ユーザーの自己分析・自己観察を支援する iOS アプリケーションです。 本アプリの開発者 (以下、 「開発者」) は、 ユーザーのプライバシーを尊重し、 収集・送信される情報の取り扱いについて、 以下の方針に従って適切に管理します。

1. 基本機能で収集する情報

本アプリの無料機能 (シナリオ選択・自己観察結果・8 軸分析・MBTI 推定・本棚提案など) は、 すべて端末内で完結します。 これらの機能では、 以下の情報を 一切外部に送信しません:

  • 氏名・メールアドレス・電話番号などの個人情報
  • 位置情報
  • 連絡先・写真・カレンダーなどの端末内データ
  • 利用ログ・解析データ・広告 ID

2. 端末内に保存される情報

本アプリは、 ユーザーの選択履歴 (どの選択肢を選んだか、 何秒考えたか、 等) を端末内のみに保存します。 これらのデータは、 ユーザー自身の自己観察のために利用され、 外部サーバーに送信されることはありません。 アプリを削除すると、 これらのデータも削除されます。

3. Pro 版 AI 深掘り対話機能 (有料機能)

本アプリの Pro / Super Pro プラン (有料サブスクリプション) をご利用の場合、 「AI と深掘りする」 機能を使うと、 入力したテキストおよび自己観察の文脈情報が外部の AI サービスに送信されます。 本機能は 有料機能であり、 ユーザーが明示的に利用を選択した場合のみ動作します。 無料プランで本機能を利用することはできません。

3-1. 送信される情報

ユーザーが AI 深掘り対話機能を利用した際、 以下の情報が外部に送信されます:

  • ユーザーが対話画面に入力したテキスト
  • 本アプリ内で実施した自己観察の結果 (MBTI 推定タイプ、 8 軸スコア、 シナリオごとの選択履歴 + 選択所要時間)
  • 端末識別子 (iOS の identifierForVendor、 月次の利用回数管理目的のみで利用)

氏名・メールアドレス・電話番号・位置情報・連絡先などの個人情報は 送信されません。 送信される自己観察情報は本アプリ内で生成されたものに限られます。

3-2. 送信経路 (中継サーバー)

ユーザーの入力は、 開発者が運用する Cloudflare Workers 上の中継サーバー を経由して、 Anthropic 社の Claude API に転送されます。 本中継サーバーは API キーを安全に保持し、 端末からのリクエストを認証する目的のみに使用されます。

  • 送信内容は中継サーバーに永続的に保存されません
  • 月次利用回数管理のため、 端末識別子と月の利用回数のみ Cloudflare Workers KV に最大 32 日間保存され、 自動削除されます
  • 中継サーバーは Cloudflare の Edge ネットワーク上で動作します (運用者: 開発者)

3-3. Anthropic 社による処理

中継サーバーから転送された情報は、 Anthropic, PBC が提供する Claude API (claude-opus-4-7 等) によって処理され、 対話の応答が生成されます。 Anthropic 社のデータ取り扱いについては、 同社のプライバシーポリシーをご参照ください:
https://www.anthropic.com/privacy

本アプリは、 Anthropic API への送信時に Prompt Caching 機能を利用しています。 Anthropic 社は API リクエストデータをモデルの学習に利用しないことを明示しています (上記プライバシーポリシー参照)。

3-4. ユーザーの選択肢

  • 本機能を 利用しない選択 はいつでも可能です (無料プランのまま自己観察結果のみご利用ください)
  • サブスクリプションは App Store の登録ページからいつでも解約できます
  • 対話履歴の端末内保存については、 アプリを削除すると共に削除されます

5. 性格タイプ分類について

本アプリは、 16 タイプの性格分類 (内向/外向、 直観/感覚、 思考/感情、 判断/知覚) に基づく、 一般的な自己観察ツールです。 公式の心理テストではありません。 本アプリの結果は、 専門的な診断や医療目的での使用を意図したものではありません。

MBTI® は The Myers-Briggs Company の登録商標です。 本アプリは MBTI 公式テストとは一切関係ありません。

7. プライバシーポリシーの変更

本ポリシーは、 必要に応じて更新されることがあります。 重要な変更がある場合は、 本ページにて告知します。

Kailua Privacy Policy

Last updated: 2026-05-08

Introduction

Kailua (hereinafter, "the App") is an iOS application that supports users' self-analysis and self-observation. The developer of the App (hereinafter, "the Developer") respects users' privacy and manages information that is collected or transmitted in accordance with the policy described below.

1. Information collected by the basic (free) features

The free features of the App (scenario selection, self-observation results, 8-axis analysis, MBTI-style type estimation, bookshelf suggestions, etc.) run entirely on the device. These features do not transmit any of the following information externally:

  • Personal information such as name, email address, or phone number
  • Location information
  • On-device data such as contacts, photos, or calendar
  • Usage logs, analytics data, or advertising identifiers

2. Information stored on the device

The App stores users' choice history (which option was selected, how long the user thought about it, etc.) on the device only. These data are used solely for the user's own self-observation and are never transmitted to any external server. Deleting the App also deletes these data.

3. Pro version AI dialogue feature (paid feature)

If you are using the Pro / Super Pro plan (paid auto-renewable subscription), when you use the "Deepen with AI" feature, the text you input together with relevant self-observation context is transmitted to an external AI service. This is a paid feature that operates only when the user explicitly chooses to use it. This feature is not available on the free plan.

3-1. Information transmitted

When the user uses the AI dialogue feature, the following information is transmitted externally:

  • Text the user has entered into the dialogue screen
  • Self-observation results generated within the App (estimated MBTI-style type, 8-axis scores, per-scenario choice history with selection times)
  • Device identifier (iOS identifierForVendor, used only for monthly usage-count management)

Personal information such as name, email address, phone number, location, or contacts is not transmitted. The transmitted self-observation information is limited to data generated within the App.

3-2. Transmission path (relay server)

User input is transmitted to Anthropic's Claude API via a Cloudflare Workers relay server operated by the Developer. The relay server is used solely for securely holding the API key and authenticating requests from the device.

  • The transmitted content is not persistently stored on the relay server
  • For monthly usage-count management, the device identifier and the month's usage count are stored in Cloudflare Workers KV for up to 32 days and then automatically deleted
  • The relay server runs on Cloudflare's Edge network (operated by the Developer)

3-3. Processing by Anthropic

Information forwarded from the relay server is processed by the Claude API (claude-opus-4-7 and similar models) provided by Anthropic, PBC to generate dialogue responses. For Anthropic's data handling, please refer to Anthropic's privacy policy:
https://www.anthropic.com/privacy

The App uses the Prompt Caching feature when sending requests to the Anthropic API. Anthropic explicitly states that API request data is not used to train its models (see the privacy policy linked above).

3-4. User choices

  • You may choose not to use this feature at any time (you can continue using only the on-device self-observation features on the free plan)
  • You can cancel your subscription at any time from the App Store subscription management page
  • Dialogue history stored on the device is deleted together with the App when you uninstall it

5. About the personality-type classification

The App is a general-purpose self-observation tool based on a 16-type personality classification (introversion/extraversion, intuition/sensing, thinking/feeling, judging/perceiving). It is not an official psychological test. Results from the App are not intended for professional diagnosis or medical use.

MBTI® is a registered trademark of The Myers-Briggs Company. The App has no affiliation whatsoever with the official MBTI test.

6. Use by children

The App is intended for users aged 4 and above. If a child uses the App, please do so with the consent of a parent or guardian. The free features do not collect any personal information. Because the Pro AI dialogue feature transmits text entered by the user to an external service, parental judgment is required when a child uses this feature.
