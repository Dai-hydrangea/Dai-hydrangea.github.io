A.L.I.S. プライバシーポリシー

最終更新日：2026年4月19日

A.L.I.S.（以下「本アプリ」）は、瀬戸大作（以下「開発者」）により提供される対話型AIアプリケーションです。本アプリはユーザーのプライバシーを最優先に設計されています。

1. 収集する情報

音声データ

マイクから取得した音声は、文字起こしの目的でのみ使用されます。音声認識は原則 iPhone 内蔵の機能を使用し、端末内で処理されます。音声データ自体は外部サーバーに送信されず、端末内にも保存されません。

テキスト化された会話内容

使用する「人格」により扱いが異なります：

  • おばあちゃん人格：Apple Intelligence（Foundation Models）を使用し、完全に端末内で処理されます。通信は発生しません。
  • ドクターX・R.MIKE・最高級・カスタム人格：Anthropic Claude API に送信されます。
  • AI 人格：Google Gemini API に送信されます。

プライバシーモード

設定で有効化すると、AI API に送信する前に電話番号・メールアドレス・人名などの個人情報を端末内で自動マスキングします。おばあちゃん人格は元々端末内処理のため、常に安全です。

会話履歴・議事録データ

すべて端末内にのみ保存されます。開発者のサーバーには送信されません。アプリを削除すると完全に削除されます。

課金情報

購入情報は Apple の App Store / StoreKit を通じて処理されます。開発者はクレジットカード情報等を直接取得しません。

2. 第三者への提供

以下のサービスに対してのみ、会話内容を送信する場合があります：

サービス用途提供会社
Claude APIドクターX・R.MIKE・最高級・カスタム人格の応答生成Anthropic, PBC
Gemini APIAI 人格の応答生成Google LLC

Anthropic のプライバシー：anthropic.com/privacy
Google のプライバシー：policies.google.com/privacy

これら以外の第三者に対するデータ送信は行いません。広告ネットワーク・アナリティクスツール等は一切使用していません。

3. データの削除

  • 設定画面内「履歴をクリア」で会話履歴を全削除できます
  • 議事録履歴は「過去の議事録」画面から個別削除可能です
  • アプリのアンインストールにより全データが完全削除されます

4. 年齢制限

本アプリは12歳以上を対象としています。未成年のご利用は保護者の監督のもとでお願いします。

キャラメイク機能には、犯罪の教示・未成年への性的表現・特定個人への誹謗・自傷の推奨・差別的表現を含む入力を拒否する安全フィルタが組み込まれています。

5. AI が生成する応答について

本アプリの応答は AI により生成されたものです。医療・法律・金融・緊急事態対応など重要な判断の根拠としては使用しないでください。AI は思考のパートナーであり、専門家の代替ではありません。
